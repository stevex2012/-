## 从全世界不同地区测试网站速度
### 地址：[dotcom-monitor](https://www.dotcom-tools.com/website-speed-test.aspx)
### 特点：
* 测试网站和网页的速度和检测性能问题从世界各地
* 所有页面元素的基于浏览器的加载时间测试
* 慢/缺失的元素的检测
* 可以选择不同浏览器，ios，Android测试pc和h5的表现
* 统计成瀑布报表，图表和图形
* 测试结果来自近20多个全球地点
